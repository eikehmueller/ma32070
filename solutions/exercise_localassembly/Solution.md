## Error norm
The following table shows the $L_2$ norm of the error for polynomial degrees $p=1,2,3,4$

| degree $p$ | error norm $\|e_h\|_{L_2(\widehat{K})}$ |
| ---------- | ---------------------------- |
|         1  |          $9.06\cdot 10^{-2}$ |
|         2  |          $2.75\cdot 10^{-2}$ |
|         3  |          $8.38\cdot 10^{-3}$ |
|         4  |          $2.82\cdot 10^{-3}$ |
